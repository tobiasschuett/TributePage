Introduction

In contrast to direct writing programs like Microsoft Word or Open Office, Latex is a rather indirekt writing program. In Latex the Document is built by programming the layout in a from the document seperate editor.

The main advantages of Latex over any direct writing program are:

  • open sourced and free
  • cleaner looking documents
  • controll over every position of text
  • faster and better way to write mathematical expressions
  • ...

Document setup

The basic setup to get startet looks like this:

\documentclass[12pt]{article} %%header section
\begin{document} %%beginning body section
\tableofcontents
\section{html}
\subsection{div tags}
\end{document} %%end body section

Similar to HTML it consists of a head, which is not part of the actual document. This is also the section where we will import Latex packages (see section Latex packages). After the head comes the body section where you program the document. Notice that single line comments can be created by using %%.
\tableofcontents automatically creates a structured table of contents based on the sections and subsections you created.

Basic Commands

Basic and often used Latex commands are:

\begin{Large} %%increases text size, similar to h1,h2,... in html
\end{Large}

\hrule %%creates vertical line
\pagebreak %%starts a new pagebreak
\textit{this text will be italic}
\footnote{this is the text for the footnote} %%footnote effecting last word before
\vspace{1cm} %%extra gab between paragraphs
\\ %%new line

\begin{center} %%horizontal placement, similar to the float property in CSS
\end{center}

\includegraphics[scale=0.7]{picture.jpg} %%import a picture and vary its' size

If you are looking for a specific command, you can find a lot more here

Latex packages

Packages are essential in Latex for enabling things such as spell checking, importing pictures and working with them and setting padding on the text. These are commanly used packages

\usepackage{german} %%spellchecking
\usepackage[onehalfspacing]{setspace} %%setting line height
\usepackage{graphicx} %%working with pictures

\usepackage[utf8]{inputenc} %%defining the encoding
\usepackage[pdftex]{hyperref} %%creating hyperlinks
\usepackage{url} %%helping Latex understand when text is a url
\usepackage[left=35mm,right=35mm,top=30mm,bottom=45mm]{geometry} %%setting the padding

If you're looking for a specif one, there is a list here

Math in Latex

Probably the major advantage has Latex when being used for scientific projects. This is becuase of its' math mode, which creates clean looking equations and are programmable to its' core. This is an example of a equation written with Latex

$$F_{G}=G*\frac{m_{1}*m_{2}}{r^{2}}$$

Let's break it down

$$ %%starts the math mode, use single $ for inline equation

F_{G} %%makes G as a subscribt of F
\frac %%creates a fraction between the first {} and the second {}
{m_{1}*m_{2}}{r^{2}}

$$ %%ends the math mode
